## 🚀 Как запустить:

```bash
# Терминал 1 - бэкенд
cd /Users/user/Downloads/food-allergen-system
./mvnw spring-boot:run

# Терминал 2 - фронтенд
cd /Users/user/Downloads/food-allergen-system/frontend-vue
npm run dev
```

Откройте браузер на **http://localhost:5173**
